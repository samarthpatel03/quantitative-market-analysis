# Quantitative Market Analysis

A machine learning project focused on predicting stock market movement
direction using technical indicators and quantitative analysis techniques.

## Current Project — Stock Movement Prediction
Predicting whether Reliance Industries (RELIANCE.NS) stock will go
up or down the next day using historical price data and technical indicators.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- yfinance
- Matplotlib, Seaborn

## Features Engineered
- Simple Moving Averages (SMA 20, SMA 50)
- Relative Strength Index (RSI)
- MACD & MACD Histogram
- Signal Line
- Bollinger Band Width
- Momentum
- Volatility

## Exploratory Data Analysis
- Price trend with Golden Cross & Death Cross analysis
- RSI overbought/oversold signals
- MACD Histogram momentum phases
- Daily return distribution with fat tail analysis
- Feature correlation heatmap
- Bollinger Bands with support/resistance zones

## Models (Coming Soon)

- XGBoost Classifier
